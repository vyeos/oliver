Oliver - A social media platform for devs. (Twitter + Github)

## Website

Post - When posting something, it can just be a text post or it can be linked to a github commit.
Home - Show the posts of fellow olivers. 
RSS - Can subscribe to rss to see its content in olier style. 
Explore - Shows popular posts and rss.

Posts and rss can be liked and reposted


## CLI

Post directly to the website using cli by writing directly writing or tagging a commit with the post
